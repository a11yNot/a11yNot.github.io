How To Contribute

Prerequisites

  • Please provide code for the accessibility issue.
  • Code contributions via Gists are preferred, we suggest creating a GitHub Account if you don't already have one.
    • BONUS! You can use the GitHub account to chat on the a11ySlackers Gitter Chatroom!

Send Us Your Not

  1. Create a Gist
  2. Tweet the Gist ID or a link to the Gist to @a11yNot

Create an Issue on our GitHub Repository

  1. Create an issue in the a11yNot Repository
  2. Suggest a Not in prose OR let us know about an accessibility issue on a11yNot.com
