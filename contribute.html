What is an a11yNot?

An a11yNot is something on a webpage that creates a barrier for people with disabilities.

If you encounter an a11yNot, we'd love to add it to our site.

How To Contribute

Prerequisites

  • Please provide code for the accessibility issue.
  • All code will be included via Gists, so create a GitHub Account if you don't already have one.

Send Us Your Nots

  1. Create a Gist
  2. Tweet a link to the Gist or the Gist ID to @a11yNot
  3. We will Reply if more information is needed and we'll Favorite your Tweet if we add it to the site!

Create an Issue on GitHub

  1. Create an issue in the a11yNot Repository
  2. Provide code for an accessibility issue.
  3. (Optional) Describe the issue.

Use GitHub To Contribute More Directly (work in progress)

  1. Check out this handy guide for Contributing to Open Source on GitHub
  2. Create a new Not using the a11yNotTemplate.md
  3. Submit a Pull Request
